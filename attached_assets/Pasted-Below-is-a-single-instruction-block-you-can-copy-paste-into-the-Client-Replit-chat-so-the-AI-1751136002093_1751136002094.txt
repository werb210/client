🔸 GOAL

After finishing components and routes:

#############################################

### What will happen

* Replit AI will create `Landing.tsx`, `NavBar.tsx`, update routes, and set `/` to this new page.
* All CTAs route to `/login`, so the user immediately enters the secure email + password + OTP flow you’ve just enabled on the staff side.
* Once authenticated, the existing protected 7-step flow works exactly as before.

Copy the block above into the **Client** project chat, let Replit run it, and you’ll have the duplicate landing page with the correct links.
