# ✅ Migration Instructions: Enforce Modern Style Guide in V2 Client

---

## 🟨 1. GLOBAL STYLE SETUP

### 🔹 What to do:

* **Ensure the style guide is globally applied** via `tailwind.config.ts` and `index.css`

### 🔹 Replit Instructions:

1. **Tailwind Theme Tokens**
   Update `tailwind.config.ts` in the root of the V2 client with your branding tokens:

   ```ts
   theme: {
     extend: {
       colors: {
         primary: '#0F766E',       // Boreal Teal
         accent: '#F97316',        // Boreal Orange
         surface: '#F8FAFC',
         onSurface: '#1E293B',
       },
       fontFamily: {
         sans: ['Inter', 'sans-serif'],
       },
       borderRadius: {
         xl: '1rem',
         '2xl': '1.5rem',
       },
     }
   }
   ```

2. **Global CSS Import**
   Ensure `client/src/index.css` includes:

   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;

   body {
     @apply font-sans bg-surface text-onSurface;
   }
   ```

3. **Layout Wrapper**
   Create `components/layout/MainLayout.tsx`:

   ```tsx
   export default function MainLayout({ children }) {
     return (
       <div className="min-h-screen bg-surface text-onSurface font-sans">
         <header className="py-4 px-6 border-b border-gray-200 bg-white shadow-sm">...</header>
         <main className="p-4 md:p-8">{children}</main>
         <footer className="text-sm text-gray-400 mt-8 text-center">© Boreal Financial</footer>
       </div>
     );
   }
   ```
