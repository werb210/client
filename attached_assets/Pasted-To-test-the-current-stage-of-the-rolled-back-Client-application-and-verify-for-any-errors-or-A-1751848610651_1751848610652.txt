## ‚úÖ 3. **Verify All API Calls to Staff App**

### A. Open Developer Tools ‚Üí Network Tab

* Reload the app
* Watch requests going to `https://staffportal.replit.app/api/...`

### B. Check for:

| Symptom                     | Meaning                                  |
| --------------------------- | ---------------------------------------- |
| `401 Unauthorized`          | Auth token missing or expired            |
| `403 Forbidden`             | Role-based rejection (e.g., wrong token) |
| `404 Not Found`             | Wrong endpoint URL (check API path)      |
| `500 Internal Server Error` | Backend is crashing                      |
| `CORS` error                | Frontend and backend domains mismatch    |

| Client App Action    | API Call to Check              |
| -------------------- | ------------------------------ |

### B. Manually test in `curl` or Postman:

